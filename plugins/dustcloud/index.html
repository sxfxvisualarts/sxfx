SXFX Dust Cloud

Particles

About This Plugin

SXFX Dust Cloud creates immersive 3D particle clouds with dynamic depth and ethereal movement. Perfect for atmospheric visuals, ambient backgrounds, and dreamlike sequences. The particles respond to wind dynamics and create a sense of depth that draws viewers into the visual experience.

Key Features

  • 3D particle system with depth perception
  • Dynamic wind and turbulence controls
  • Adjustable particle density and size
  • Depth-based blur and fog
  • Color gradient support
  • Image Mask input for shape control
  • Performance optimized rendering

Technical Specifications

Format FFGL 2.x
Platform Windows / macOS
Compatible Resolume Arena & Avenue
GPU OpenGL 4.1+
