SXFX Parallaxis

Warping

About This Plugin

SXFX Parallaxis is a multi-layer parallax engine that adds incredible depth perception to any visual. Create smooth motion trails, layer separation effects, and depth blur that transforms flat visuals into immersive 3D-like experiences. Perfect for adding dimension to logos, text, and any source.

Key Features

  • Multi-layer depth separation
  • Smooth motion trails with decay
  • Depth-based blur effect
  • Adjustable parallax intensity
  • Works with any input source
  • Real-time performance optimized

Technical Specifications

Format FFGL 2.x
Platform Windows / macOS
Compatible Resolume Arena & Avenue
GPU OpenGL 4.1+
