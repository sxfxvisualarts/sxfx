About This Plugin

SXFX Block Madtrix VU is a powerful geometric evolution engine that merges 2D procedural filling with 3D raymarching structures. Designed for high-impact live performances, it features advanced audio reactivity and thermal color dynamics.

Key Features

  • Fill Grid (2D): 13 algorithms for procedural organic filling
  • Wall (3D): Deep raymarching structures with volumetric lighting
  • Puzzle: Dynamic sliding block engine with bevel effects
  • VU Mode: Integrated 8-band spectrum analyzer reactivity
  • Smoothing time controls and flicker progression
  • Thermal color gradients and real depth perception

Technical Specifications

Format FFGL 2.x
Platform Windows (contact for MAC)
Compatible Resolume Arena & Avenue
GPU OpenGL 4.1+
