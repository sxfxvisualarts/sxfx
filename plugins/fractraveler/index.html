About This Plugin

SXFX Fractraveler takes your audience on a journey through infinite fractal dimensions. Create hypnotic visuals with endless zoom capabilities, multiple fractal types, and stunning color palettes. The perfect tool for psychedelic visuals, ambient environments, and mind-bending effects.

Key Features

  • Infinite zoom with seamless looping
  • Multiple fractal types (Mandelbrot, Julia, etc.)
  • Customizable color palettes
  • Zoom speed and direction controls
  • Rotation and distortion effects
  • GPU-accelerated rendering

Technical Specifications

Format FFGL 2.x
Platform Windows (contact for MAC)
Compatible Resolume Arena & Avenue
GPU OpenGL 4.1+
